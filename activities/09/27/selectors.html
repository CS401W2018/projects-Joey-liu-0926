What is the difference between an ID and a Class

An ID is unique and should be used only once per page. Each element can have only one ID.

A class can be reused on multiple elements. Multiple elements can share the same class, and an element can have more than one class.

What is the difference between a SPAN tag and a DIV tag?

Div Block-level element, used for larger content blocks and layout structuring, takes up full width and starts on a new line. Spin Inline element, used for styling small parts of text or content within larger elements, remains in the flow of text.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

div: Generic block-level container, used for grouping content but provides no semantic meaning. header: Defines a header section, often for navigational or introductory content. main: Represents the main content of the page, should only be used once per page. footer: Defines a footer section, usually for closing information, links, or contact details.
